天地图WMTS服务接口文档

一、概述

本接口集合为天地图WMTS（Web Map Tile Service）服务提供支持，涵盖组图层、单图层以及图层与图层组叠加的请求处理功能，以满足多样化的地图数据获取需求。

二、接口列表

（一）WMTS请求接口

  • 接口路径：
    • /geoair/mix/{layerName}/wmts
  • 功能描述：处理组图层的WMTS请求，根据请求参数获取相应的地图瓦片或能力信息等。
  • 参数说明：
    • {layerName}：必填，图层名称，用于指定请求的目标图层 。
    • request（隐式参数，通过HttpServletRequest获取）：包含请求头、请求参数等信息，用于获取具体的请求细节，如请求类型（GetCapabilities、GetTile等）、瓦片编号等。
  • 请求示例： 
    http://XXX.XXX.XXX.XXX:26888/geowebcache/geoair/mix/{layerName}/wmts?service=WMTS&request=GetTile&version=1.0.0&layer={layerName}&style=&Format=image/png&TileMatrixSet=EPSG:4490&TileMatrix={z}&TileRow={y}&TileCol={x}

（二）XYZ请求接口

  • 接口路径：
    • geoair/xyz/{layerName}/4490/{z}/{x}/{y}?zxyType=zxy&orginType=wmts
  • 功能描述对于图层的XYZ坐标系请求接口，用于根据xyz获取指定图层瓦片数据。
  • 参数说明：
    • {layerName}：必填，图层名称，用于指定请求的目标图层。
    • {z}：必填，图层名称，z级别。
    • {x}：必填，图层名称，x级别。
    • {y}：必填，图层名称，y级别。
    • {zxyType}：选填，标识你的请求是zxy发过来的还是zyx发过来的，默认是zxy。
    • {orginType}：选填，标识你的请求方式是xyz请求还是tms请求，默认是xyz（wmts）。
  • 请求示例： 
    http://XXX.XXX.XXX.XXX:26888/geowebcache/geoair/xyz/abcdLayer/4490/4/15/1?zxyType=zxy&orginType=wmts

（三）TMS请求接口

  • 接口路径：
    • geoair/xyz/{layerName}/4490/{z}/{x}/{y}?zxyType=zxy&orginType=tms
  • 功能描述 与2接口一致，就是你需要把orginType 参数换成 tms
  • 参数说明：
    • {layerName}：必填，图层名称，用于指定请求的目标图层。
    • {z}：必填，图层名称，z级别。
    • {x}：必填，图层名称，x级别。
    • {y}：必填，图层名称，y级别。
    • {zxyType}：选填，标识你的请求是zxy发过来的还是zyx发过来的，默认是zxy。
    • {orginType}：选填，标识你的请求方式是xyz请求还是tms请求，默认是xyz（wmts）。
  • 请求示例： 
    http://XXX.XXX.XXX.XXX:26888/geowebcache/geoair/xyz/abcdLayer/4490/4/15/1?zxyType=zxy&orginType=tms

（四）调试网格WMTS接口

  • 接口路径：
    • /debug/grid/wmts
  • 功能描述： 用于调试网格坐标系的WMTS服务，支持返回带有网格线的地图瓦片，主要用于开发阶段的坐标系校准、瓦片切割范围验证等场景。
  • 参数说明：
    • service：必填，固定值 WMTS，指定服务类型。
    • request：必填，支持 GetTile（获取带网格的瓦片）和 GetCapabilities（获取调试网格服务能力）。
    • version：必填，固定值 1.0.0，指定WMTS协议版本。
    • layer：必填，固定值 grid，标识调试网格图层。
    • style：可选，任意值 c
    • Format：必填，固定值 image/png，返回瓦片格式。
    • TileMatrixSet：必填，固定值 c
    • TileMatrix：必填，瓦片层级（如 0 表示第0层级）。
    • TileRow/ TileCol：必填，瓦片行列号（从0开始计数）。
  • 请求示例： 
    http://127.0.0.1:26888/geowebcache/debug/grid/wmts?service=WMTS&request=GetTile&version=1.0.0&layer=grid&style=c&Format=image/png&TileMatrixSet=EPSG:4490&TileMatrix={z}&TileRow={y}&TileCol={x}

（五）调试网格XYZ/TMS接口

  • 接口路径：
    • g/geoair/debug/grid/xyz/{layerName}/{gridSrid}/{z}/{x}/{y}?zxyType=zxy&orginType=tms
  • 功能描述 支持返回带有网格线的地图瓦片，主要用于开发阶段的坐标系校准、瓦片切割范围验证等场景
  • 参数说明：
    • {layerName}：必填，图层名称，用于指定请求的目标图层。
    • {gridSrid}：必填，你的网格坐标系，如4490 ， 主要用于你的坐标原点为tms的时候，我来判断该怎么翻转。
    • {z}：必填，图层名称，z级别。
    • {x}：必填，图层名称，x级别。
    • {y}：必填，图层名称，y级别。
    • {zxyType}：选填，标识你的请求是zxy发过来的还是zyx发过来的，默认是zxy。
    • {orginType}：选填，标识你的请求方式是xyz请求还是tms请求，默认是xyz（wmts）。
  • 请求示例： 
    http://XXX.XXX.XXX.XXX:26888/geowebcache/geoair/debug/grid/xyz/yeshi2024:4_3_2_1/4490/4/10/4?zxyType=zxy&orginType=tms
